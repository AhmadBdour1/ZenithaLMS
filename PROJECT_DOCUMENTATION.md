# وثيقة مشروع ZenithaLMS

## نظرة عامة

**ZenithaLMS** هو نظام إدارة تعلم متقدم ومتطور بالذكاء الاصطناعي، مبني باستخدام Laravel 12 و PHP 8.4+ مع تقنيات الويب الحديثة. يوفر حلًا شاملاً للتعليم الإلكتروني مع ميزات متقدمة تشمل التوصيات المدعومة بالذكاء الاصطناعي، والتعلم التكيفي، والهندسة متعددة المستأجرين، والتحليلات الشاملة.

## المعلومات الأساسية

- **اسم المشروع**: ZenithaLMS
- **الإصدار**: 1.0.0

- **قاعدة البيانات**: MySQL 8.0+ / PostgreSQL 12+ / SQLite 3.8+

- **Playwright**: 1.58.2+ (اختبارات المتصفح)
- **Docker**: للحاويات والنشر

## التثبيت والتكوين

### المتطلبات
- PHP 8.2+
- MySQL 8.0+ أو PostgreSQL 12+
- خادم الويب: Apache أو Nginx
- ملحقات PHP: PDO, MBString, Tokenizer, XML, CTYPE, JSON, BCMath, FileInfo, OpenSSL

### خطوات التثبيت
1. استنساخ المستودع
2. تثبيت الحزم (`composer install`, `npm install`)
3. تكوين البيئة (`cp .env.example .env`, `php artisan key:generate`)
4. إعداد قاعدة البيانات (`php artisan migrate`, `php artisan db:seed`)
5. بناء الأصول (`npm run build`)
6. تشغيل التطبيق (`php artisan serve`)
